# Portofolio Data Engineering

Portofolio ini berisi kumpulan project data engineering yang berfokus pada penyelesaian masalah data di dunia nyata, termasuk pembuatan data pipeline, pembersihan data (data cleaning), dan pemodelan data menggunakan Python dan SQL.

Tujuannya adalah untuk mensimulasikan bagaimana data bisnis yang awalnya mentah dan tidak terstruktur dapat diolah menjadi dataset yang bersih, terstruktur, dan siap digunakan untuk analisis serta pengambilan keputusan.

---

## 🔧 Tech Stack
- Python (Pandas, NumPy)
- SQL
- Excel
- Konsep Data Modeling (Fact & Dimension Table - basic)
- Pengolahan data CSV / dataset terstruktur

---

## 📁 Project

### 1. Sistem Data Transaksi Sintetis
Project ini mensimulasikan data transaksi bisnis yang menyerupai kondisi nyata untuk memahami bagaimana data operasional berkembang dari waktu ke waktu.

**Highlight:**
- Membuat dataset transaksi multi-periode (10k – 100k baris)
- Mensimulasikan perilaku bisnis seperti penambahan menu dan pertumbuhan revenue
- Membuat pola data yang realistis untuk kebutuhan analisis

**Tools:**
Python, Pandas, Logika Bisnis, Simulasi Data

---

### 2. Pipeline Pembersihan & Transformasi Data
Project ini berfokus pada pembersihan dataset yang memiliki missing value, inkonsistensi, dan struktur yang tidak rapi.

**Highlight:**
- Menangani missing value dan data tidak konsisten
- Standarisasi dan pembersihan data transaksi
- Mengubah data mentah menjadi format yang siap analisis

**Tools:**
Python, Pandas, Data Cleaning, Data Wrangling

---

## 📊 Pembelajaran yang Didapat
- Data di dunia nyata hampir selalu tidak bersih dan tidak lengkap
- Data cleaning adalah bagian penting dalam data pipeline
- Struktur data yang baik sangat mempengaruhi kualitas analisis
- Dasar pemahaman tentang alur data dalam sistem bisnis

---

## 🚀 Pengembangan Selanjutnya
- Membangun pipeline ingestion berbasis API
- Menyimpan data ke database (PostgreSQL / MySQL)
- Automasi pipeline ETL sederhana
- Mendalami data modeling (star schema / dimensional modeling)
